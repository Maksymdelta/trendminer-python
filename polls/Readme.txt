The Mapper_Tweet.py is for calculating the sentiment ratio for three parties, namely, Conservative, Labour, and Liberal Democrats.
     The input are the file lists: tweets_pre1_list, tweets_pre2_list, tweets_pre3_list
     The output will be: tweets_pre1_list.out, tweets_pre2_list.out, tweets_pre3_list.out
     The command is: python Mapper_Tweet.py tweets_pre1_list
                     python Mapper_Tweet.py tweets_pre2_list
                     python Mapper_Tweet.py tweets_pre3_list


The Reducer_Tweet.py is for sorting the output of Mapper_Tweet.py and stored them into three separate files.
     The output is: results_con, results_lab, results_lib


Each step has been commented in detil in the codes.
